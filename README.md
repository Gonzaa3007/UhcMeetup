
## UhcMeetup

A very customizable UhcMeetup plugin for current Minecraft Versions!




## Features

- MC Version: 1.17.x
- Plugin Version: 1.2
- Stats & Leaderboard
- Scenarios & Vote Scenarios system
- MySQL compatibility
- Customizable messages, config, scoreboard and kits
- Random kits


## Commands
`/vote`
- Open the Vote GUI

`/stats <player>`
- See a player stats

`/topstats`
- Open the TopStats GUI

`/uhcmeetup setlobby`
- Sets the spawn

`/uhcmeetup reload`
- Reload all configs

`/forcestart`
- Starts the Meetup

## Permissions

- meetup.admin
- meetup.vote
- meetup.forcestart
