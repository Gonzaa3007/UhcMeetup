# Features
* UHC Meetups Automatics
* Customizable messages, config, scoreboard and kits
* Vote Scenarios system
* Stats system
* Random Kits

# Commands
```/vote```
- Open the Vote GUI

```/stats <player>```
- Open the Stats GUI

```/topstats```
- Open the TopStats GUI

```/uhcmeetup setlobby```
- Sets the spawn

```/uhcmeetup reload```
- Reload all configs

```/forcestart```
- Starts a Meetup
# Permissions
- meetup.admin
- meetup.vote
- meetup.forcestart
